FSharp and Azure Functions example

a day ago FSharp, Getting Started, F# Mouad Cherkaoui

this is a little tutorial demonstrating the usage of F# language, and ADO.Net as data access mechanism, and also substituting it with a new mechanism that use Azure Table Storage as store.

Prerequisites :

installing dotnetcore and vscode :

first of all our sample program is based on .Net core framework, and also we will use azure function cli tools to generate debug and deploy our function to Azure App Service.

  • go to dot.net, and then click download link.
  • install the package.
  • go to code.visualstudio.com, and download the right version for your operating system, for linux you will find instructions about installing it on this link.
  • launch vscode, and in the left sidebar you click on the plugins icon, the last one.
  • then search ionide, and install it.
